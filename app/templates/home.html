The principle behind STEMCstudio is to provide a learning environment in which the student can verify personal understanding and conceptual models by constructing a working software model. This learning environment takes the form of a general-purpose software development environment optimized for Mathematics and Computer Graphics. Programming a computer, sometimes in collaboration with others, provides a non-threatening environment, clarifies understanding, challenges mastery, and motivates further exploration.

Motivation

  • The importance of geometry to modern mathematics has grown over the last century but is not reflected in student course choices.
  • The Geometry curriculum of Euclid's elements and Descartes coordinates is obsolete, ineffective, and inefficient compared to modern approaches which unify algebra and geometry.
  • Mathematics and Physics courses under-utilize and misuse computers.

Manifesto

  • Rework the geometry curriculum around the modern mathematical notation of Geometric Algebra.
  • Ensure students learn a general purpose programming language so that they can use computers for truly active and constructive learning by programming Mathematics, Physics and Computer Graphics.

Features and Benefits of STEMCstudio

STEMCstudio is unique among browser editing environments by being optimized for learning Mathematics and Geometry.

Code Editor and Preview in your browser
No setup or installation required. Just use a compatible browser such as Chrome, Firefox or IE9.
JavaScript, HTML, CSS
Use the tools that are the bread-and-butter of a contemporary software developer. The modeling-development environment is general purpose and allows external libraries to be used and services to be called over the internet.
TypeScript Language
TypeScript is JavaScript with optional type information and features taken from future JavaScript releases. Using TypeScript provides intelligent checking and context-sensitive help, allowing the student to spend more time focusing on the task in hand instead of trying to find errors or lookup documentation.
Local Caching
Allows you to work even in situations with limited internet connectivity. Your work is automatically saved locally in the browser and can be uploaded when connectivity becomes available.
Cloud Sharing
Students and educators can store their work permanently in their personal and free GitHub account. Educators may use GitHub as a means to create and distribute assignments, while students may use GitHub to provide a permanent record of their work and as a means for sharing.
Operator Overloading
Operator Overloading is essential for making mathematical programming look natural when dealing with structured types such as vectors and matrices. STEMCstudio supports a rich set of mathematical operators (useful for performing multivector analysis). In addition to the usual arithmetic operators, there is support for inner and outer products, as well as left and right contraction. These operators may be used with your own custom datatypes. Operator Overloading is optional.
VPython-like capability
VPython creates a simple programming experience by using a high-level graphics library called visual. STEMCstudio achieves the same degree of simplicity through a library called EIGHT. The graphics abstractions in this library are similar to the popular 3D WebGL library called THREE. The name EIGHT alludes to 23, which is the number of basis elements in the corresponding geometric space.
Geometric Algebra Library
STEMCstudio provides a library, EIGHT for performing Geometric Algebra computations in 2D and 3D Euclidean geometries.
WebGL Mathematical Computer Graphics Library
STEMCstudio provides a library EIGHT that helps to manage the complexity of WebGL shader programs rather than trying to hide it. This is important for high-performance graphics and mathematical flexibility.
Units of Measure
STEMCstudio incorporates a library EIGHT that combines geometric quantities with units to create measures. Students can explore dimensional analysis and the S.I. system of units. Calculations may be performed without units or by carrying through units.
ES6 Modules
Even small authentic problems can require hundreds or thousands of lines of code. Professionals avoid the comprehension issues inherent in large monolithic code bases by employing modular decomposition. STEMCstudio allows you to break down your application into multiple re-usable files, and re-compose them automatically into an application through the ES6 module loader.
Unit Testing
A unit testing framework enables the practice of test-driven development. It also ensures that the critical parts of your code work according to specifications. Automated unit testing is supported through the popular Jasmine framework and an HTML reporter. This allows you to run repeatable test on your code and see the results in the browser.
README Documentation and LATEX support.
A README (Markdown) file is an invaluable adjunct to any coding project. It may describe any aspect of the project and is a standard file in a GitHub repository. STEMCstudio supports the README markdown file format and performs automatic real-time translation to HTML in the browser. Additionally, STEMCstudio supports mathematical LATEX markup using MathJax.

A STEM Curriculum for the 21st Century

Learning Geometric Algebra through Computational Modeling and Computer Graphics.

I am frequently asked whether there is a curriculum for learning Geometric Algebra at the middle and high-school level using STEMCstudio to practice the concepts. To address this need I have created a blog in which I will capture a possible approach. I hope this will be a proving ground for developing a new curriculum. The Geometric Physics blog will explain the pedagogic philosophy, incrementally develop the necessary mathematical theory for Geometric Algebra, provide the practical information for using STEMCstudio and related technologies, and finally guide the student or educator in developing their own geometric numbers, computations and graphics. Once the necessary mathematical and computing infrastructure has been developed, it will be applied to understand the laws of Physics which are described geometrically.

Audience

STEMCstudio is a general purpose cognitive tool for the scientific community.

STEMCstudio was conceived to tackle the problem of learning geometry effectively, but its general purpose nature makes it useful to a much wider audience. At the youngest end of the spectrum it can be used as the first introduction to a grown-up programming language. In the middle of the spectrum the tool may be used for learning and experiencing mathematical geometry with the programming being second-nature. Educators may use STEMCstudio for demonstration purposes or to set hands-on problems. Researchers may use the tool for convenient computation, exploring problems, or to present their work.
